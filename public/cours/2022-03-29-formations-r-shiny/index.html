Avec Christophe Bontemps (INRAE, Toulouse) nous avons donné un cours d’introduction à Shiny. Le cours utilise très peu de code R, et se concentre sur les fonctions spécifiques à Shiny.

Ce cours d’une journée a été donné deux fois : dans le réseau métier “Ingé+Stat” du département ECOSOCIO de l’INRAE (en 2017) et aux rencontres Ingénieurs Statisticiens de Toulouse (en 2018).
